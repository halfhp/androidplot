# Androidplot

![image](docs/images/preview.png)

A library for creating dynamic and static charts in Android apps. Itâ€™s designed from the ground up for Android, 
compatible with all versions of Android from 1.6 onward and is **used by over 
[1,000 apps](http://www.appbrain.com/stats/libraries/details/androidplot/androidplot) on Google Play**.

**Features:**

* Line Charts
* Scatter Charts
* Bar Charts
* Pie Charts
* Step Charts
* Candlestick Charts
* Dynamic plots
* Pan & Zoom
